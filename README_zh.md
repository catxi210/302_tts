## 项目特性

### ⚙️ 跨平台集成
 -  OpenAI
 -  Azure
 -  豆包
 -  FishAudio
 -  Minimax
### 🌍 多语言支持
  支持多种语言的文案生成，包括但不限于中文、英文、日文等，帮助用户轻松应对全球市场的内容创作需求。
### 🎧 播放与调节
  支持在线播放与倍速调节。
### 🗣️ 三大功能
  - 语言转文字
  - 语音转语音
  - 声音克隆
### ⏺️ 上传与下载
  支持上传音频文件或录音，也支持下载mp3文件

## 开发&部署
1. 克隆项目 `git clone https://github.com/302ai/302_tts`
2. 安装依赖 `pnpm install`
3. 配置302的API KEY 参考.env.example
4. 运行项目 `pnpm dev`
5. 打包部署 `docker build -t tts . && docker run -p 3000:3000 tts`
